Opacity Variants

White Opacity Scale (Dark Mode Overlays)

Used for creating subtle separation on dark backgrounds

0.01 (1%) Barely visible - footer content bottom
0.02 (2%) Very subtle lift
0.03 (3%) Cards, toggle section, testimonial cards
0.06 (6%) Borders, separators
0.08 (8%) Toggle container, card borders
0.12 (12%) Toggle border, prominent borders
0.15 (15%) Light mode footer toggle container
Usage Principle: Use lower opacity (1-3%) for large areas, higher opacity (8-15%) for interactive elements that need more visual weight.
